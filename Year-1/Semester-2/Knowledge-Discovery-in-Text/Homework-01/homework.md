# Homework assignment #1

## Name: Automatic question answering

Participate in **FinMMEval** competition. Follow the instructions on the official task website ([https://mbzuai-nlp.github.io/CLEF-2026-FinMMEval-Lab/](https://mbzuai-nlp.github.io/CLEF-2026-FinMMEval-Lab/)):

* Register for the task - follow the instructions and select FinMMEval lab. You are free to select others if you want to participate separately from the homework.
* Choose a task you want to participate in: Task 1: Financial Exam Q&A.
* Download the available data.
* Decide on approach(es) to try out. You can use any of the provided details in the dataset. 
* Submit your results in the official leaderboard.
* Write short description of the approach(es). The description should contain explanation of the approach you have used and steps taken to process the data. Furthermore, the results must be reported with a short discussion, especially if more than 1 approach was tested.

**Deadline:** 21th of April

**Teams are not allowed!**
**Code generation is allowed!**
**Fully generated documentation will be penalized!**

**Submission format:** submit .zip of the used code and a short (up to one page) description of your system. The description should only focus on what you have done, without unnecesary details like data stats. Report your score!
